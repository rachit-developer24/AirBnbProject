# Airbnb Clone (SwiftUI)

A modern Airbnb-style iOS app built with **SwiftUI**, focused on clean architecture, smooth navigation, and a production-style UI/UX.  
This project showcases how I build scalable SwiftUI apps using **MVVM**, reusable components, and modern iOS patterns.

- Explore listings with a clean feed-style UI
- Listing details screen (photos, info, amenities, etc.)
- Image carousel (TabView / paging)
- Map integration (MapKit) with region setup and pins 
- Search / filters UI (destination, dates, guests) 
- Smooth navigation using NavigationStack
- Reusable UI components for consistent design
- Mock data / previews for rapid UI iteration
